# LQRelayoutButton
调整UIButton的title和image位置,
在原来的基础上增加了 imageSize 设置 image 的大小,而不是之前的7/8 的大小, 同时添加了 image 的 offset to top or right
## 添加
##### 手动
将项目中的 LQRelayoutButton 文件夹添加到项目目录即可！

##### Cocoapods
```
pod 'LQRelayoutButton'
```
